## Observacoes

- O prototipo aceita `dashboard/agnes-portrait.webp` como fallback para a avo, mas o nome preferido para novas artes e `dashboard/vovo-portrait.webp`.
- Para este novo estilo, trate cada arte como parte de um brinquedo digital: menos textura, mais clareza e modularidade.
- Cenarios principais podem ser aproveitados tanto como hero visual quanto como base de telas clicaveis.
