# Bruxinha Cora

Protótipo estático em HTML, CSS e JavaScript cobrindo as fases 1, 2 e 3 do app interativo.

## Escopo desta entrega
- Dashboard com mapa afetivo, personagens e navegação principal
- Casa da Mamãe com três microinterações
- Casa da Vovó com passeio guiado, rotina e quiz de estilo
- Quarto da Cora com roupas, decoração e interação com Noir
- Jardim das Sete Luas com investigação leve, combinações mágicas e exploração
- Caderno da Cora com páginas íntimas, estados emocionais e lembranças do dia
- Bruxozap com conversas interativas entre Cora, mamãe e papai
- Roadmap visual para as próximas fases

## Como abrir
1. Abra `index.html` no navegador.
2. Navegue pelas abas ou pelos cartões do dashboard.
3. Clique no Noir, nas meias, nas perguntas e nas ações do Bruxozap.

## Artes e placeholders
- Os espaços reservados para ilustrações já estão no HTML com `data-asset-slot`.
- A organização recomendada dos arquivos está em `GUIA_TECNICO_ILUSTRACOES.md`.
- As pastas-base para assets ficam em `assets/ilustracoes/`.
